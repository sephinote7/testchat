## 상세 화면별 기능

### 1. 상담 완료 (ID: 1, 5)

- **표시 내용**:
  - 상담 내용
  - 상담사 정보
  - 채팅 상담 내역 (채팅인 경우)
  - **리뷰 작성하기** 버튼
- **클릭 시**: 리뷰 작성 모달 표시
  - 별점 선택 (1~5점)
  - 리뷰 내용 작성
  - 완료 시 → 리뷰 작성 완료 모달

### 2. 상담 예약 대기 (ID: 2)

- **표시 내용**:
  - 상담 내용
  - 상담사 정보
  - "상담사 확인 후 예약이 확정됩니다" 안내 메시지
- **버튼**: 없음 (대기 중)

### 3. 상담 예약 (완료) (ID: 3)

- **표시 내용**:
  - 상담 내용
  - 상담사 정보
  - **수정하기** + **취소하기** 버튼
- **클릭 시**:
  - 수정하기 → 상담 일정 수정 완료 모달
  - 취소하기 → 상담 취소 완료 모달
- **참고**: 예약일 1일 전까지만 수정/취소 가능
  - 1일 미만 남은 경우 → "수정/취소 불가 안내" 모달

### 4. 상담 예약 취소 (ID: 4)

- **표시 내용**:
  - 상담 내용
  - 상담사 정보
  - "상담이 취소되었습니다" 안내 메시지
  - 취소일 표시
- **버튼**: 없음

## AI 상담 테스트

### AI 상담 상태

- **상담 중**: 메시지 입력창 + "상담 완료하기" 버튼
- **상담 완료**: 메시지만 표시 (입력 불가)
