# 지도 반경 검색 가이드

## 현재 구현 (DB 기준 반경 5km)

- **현재 위치 기준 5km 이내**에 있는 **우리 DB에 등록된** 상담센터/복지센터만 자동으로 필터링되어 표시됩니다.
- 사용자가 위치 허용 시 기본이 "5km 이내" 모드이며, **전체 보기** 버튼으로 토글할 수 있습니다.
- API: `GET /api/centers?lat=...&lng=...&radiusKm=5&page=1&pageSize=7`

## 상담센터/복지센터/고용노동부 등 “외부 시설”까지 자동 검색하고 싶을 때

DB에 없는 **카카오맵에 등록된 시설**(상담센터, 복지센터, 고용노동부 등)까지 자동 검색하려면 **카카오 로컬 API**를 쓰면 됩니다.

### 방법 요약

1. **카카오 개발자 콘솔**  
   - [Kakao Developers](https://developers.kakao.com) → 앱 선택 → **로컬** API 사용 설정  
   - **REST API 키** 확인 (JavaScript 키와 별도)

2. **키워드로 장소 검색 API**  
   - `GET https://dapi.kakao.com/v2/local/search/keyword.json`  
   - 파라미터 예:  
     - `query`: `상담센터` / `복지센터` / `고용노동부` / `취업지원센터` 등  
     - `x`: 경도 (longitude)  
     - `y`: 위도 (latitude)  
     - `radius`: 반경 미터 (예: 5000 → 5km)

3. **구현 위치**  
   - REST API 키는 **백엔드에만** 두고, 백엔드에서 카카오 로컬 API를 호출한 뒤 결과를 프론트에 넘기는 방식을 권장합니다.  
   - 예: `GET /api/centers/nearby-places?lat=...&lng=...&radius=5000&keyword=상담센터`  
   - 여러 키워드(상담센터, 복지센터, 고용노동부)를 각각 호출해 결과를 합쳐서 지도/목록에 표시할 수 있습니다.

4. **지도 표시**  
   - 로컬 API로 받은 위도/경도로 카카오맵에 마커를 추가하면 됩니다 (현재 DB 센터 마커와 동일한 방식).

정리하면, **지금은 “우리 DB + 반경 5km”**만 적용되어 있고, **카카오에 있는 상담센터/복지센터/고용노동부 등까지 자동 검색**하려면 위처럼 **카카오 로컬 API**를 백엔드에서 호출하는 방식으로 확장하면 됩니다.
